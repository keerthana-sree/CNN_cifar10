# CIFAR-10 Image Classification using Convolutional Neural Network (CNN)

This project is a Deep Learning based image classification system developed using Convolutional Neural Networks (CNN) and the CIFAR-10 dataset.  
The model is trained to identify and classify images into multiple object categories.

---

## 📌 Project Overview

The main objective of this project is to build an efficient CNN model for image recognition and classification using the CIFAR-10 dataset.

The project includes:
- Dataset loading
- Image preprocessing
- CNN model building
- Model training and testing
- Accuracy evaluation
- Image prediction

---

## 🛠 Technologies Used

- Python
- TensorFlow / Keras
- NumPy
- Matplotlib
- Jupyter Notebook

---

## 📂 Dataset

Dataset Used: **CIFAR-10**

The dataset contains 60,000 color images divided into 10 classes.

Classes include:
- Airplane
- Automobile
- Bird
- Cat
- Deer
- Dog
- Frog
- Horse
- Ship
- Truck

---

## 🧠 Deep Learning Model

### Convolutional Neural Network (CNN)

Layers Used:
- Conv2D
- MaxPooling2D
- Flatten
- Dense
- Dropout

Activation Functions:
- ReLU
- Softmax

Optimizer:
- Adam

Loss Function:
- Sparse Categorical Crossentropy
